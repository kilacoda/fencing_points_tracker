Fencing Points Tracker

This page is meant to simplify looking up Olympic qualification standings for the 2024 Paris Olympics. You can select different options in the dropdowns below to get the table for a specific event.

The data is scraped from the FIE website, and is updated whenever I make a change to the code or likely after a major event. If you want to see the code, or contribute, check out the Github repo: Source (GitHub)
